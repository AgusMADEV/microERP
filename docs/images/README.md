# Imágenes de Documentación

Esta carpeta contiene las capturas de pantalla utilizadas en el README principal.

## Capturas Requeridas

Para completar la documentación visual del proyecto, añade las siguientes imágenes:

1. **login.png** - Pantalla de inicio de sesión
   - Muestra el formulario de login con los gradientes
   - Resolución recomendada: 1200x800px

2. **dashboard.png** - Vista principal del dashboard
   - Navegación lateral con lista de tablas
   - Contenido principal mostrando registros
   - Resolución recomendada: 1920x1080px

3. **custom-form.png** - Formulario personalizado
   - Ejemplo de un template personalizado (clientes/customers)
   - Muestra los campos estilizados
   - Resolución recomendada: 1200x900px

4. **card-view.png** - Vista de tarjetas
   - Registros mostrados en formato card
   - Múltiples tarjetas visibles
   - Resolución recomendada: 1400x900px

5. **charts.png** - Gráficos y reportes
   - Gráfico donut con leyenda
   - Vista del modo reporte
   - Resolución recomendada: 1200x800px

## Cómo Tomar las Capturas

1. Navega a cada sección del microERP
2. Usa una herramienta de captura de pantalla (Windows Snipping Tool, macOS Screenshot, o similar)
3. Guarda cada imagen con el nombre exacto indicado arriba
4. Coloca las imágenes en esta carpeta

## Formato

- **Formato**: PNG (recomendado) o JPG
- **Calidad**: Alta resolución pero optimizada para web
- **Peso máximo**: ~500KB por imagen (comprime si es necesario)

## Herramientas de Optimización

- [TinyPNG](https://tinypng.com/) - Compresor de imágenes PNG/JPG
- [Squoosh](https://squoosh.app/) - Optimizador de imágenes web
- [ImageOptim](https://imageoptim.com/) - App para macOS

---

**Nota**: Mientras no tengas las imágenes, el README mostrará enlaces rotos para las imágenes. Esto es normal y se resolverá cuando añadas las capturas.
